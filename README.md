It is a typescript backend server with mongoose.

if you want to run it locally , you have to install all dependency at first .
Then have a commad npm run start:dev this command run the application locally

all dependency list : 
"bcrypt"
    "cors"
    "dotenv"
    "express"
    "mongoose"
    typescript"
    "@types/bcrypt"
    "@types/cors"
    "@types/express"
